# Privacy Policy

**Last updated:** June 2026

## Overview

BuildProof ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and otherwise handle your information when you use our website and services.

## Information We Collect

### Information You Provide

- Account information: email, name, and profile details.
- Project information: website URLs, repository URLs, builder used, backend used, and launch status.
- Scan data: temporary project code and configuration files when a deeper scan is requested.

### Information Collected Automatically

- Device information: IP address, browser type, operating system.
- Usage information: pages visited, time spent, actions taken.
- Analytics: general usage patterns via privacy-conscious analytics tools where enabled.

## How We Use Your Information

We use collected information to:

1. Provide services: run audits and generate reports.
2. Improve products: understand how users interact with BuildProof.
3. Communicate: send important service updates or security alerts.
4. Protect the service: detect and prevent fraud or abuse.
5. Comply with legal and regulatory obligations.

## Data Processing and Storage

### Raw Code Handling

- Raw code is processed temporarily for scans only.
- Raw code is deleted after scanning unless you explicitly opt into storage.
- Stored snippets are redacted of sensitive information.
- Raw code is never sold or shared for unrelated third-party use.

### Secure Processing

- Scans run in isolated temporary environments.
- Service keys are not stored in scan results.
- API keys are redacted from all stored findings.
- Credentials are never exposed in reports.

## Data Retention

- Scan results: stored for audit and reporting purposes until deleted by the user or retention policy.
- Raw code: deleted after processing unless archived with user consent.
- Payment data: retained per Stripe and legal accounting requirements.
- Usage logs: retained for security and operational purposes, normally 90 days unless needed for incident investigation.

## Your Rights

You have the right to:

- Access your personal data and projects.
- Correct inaccurate information.
- Delete your account and associated data.
- Export your data in standard formats.
- Withdraw consent for data processing where consent is the legal basis.
- Lodge complaints with relevant authorities.

## Data Deletion

To delete your account and data:

1. Go to account settings.
2. Select "Delete my account".
3. Confirm the action.
4. Your data will be permanently deleted within 30 days unless retention is legally required.

## Security

We implement security measures including:

- HTTPS encryption for all communications.
- Firebase Security Rules for Firestore and Storage.
- Server-side ownership checks for privileged operations.
- Secret key management best practices.
- Regular security audits and reviews.
- No plaintext password storage.
- Secure authentication flows.

No security system is 100% secure. Report vulnerabilities responsibly.

## Compliance

### GDPR (EU/UK Users)

We aim to comply with the UK GDPR and EU GDPR, including:

- Lawful basis for processing.
- Data protection impact assessment where appropriate.
- Data processing agreements with vendors where required.
- User rights as outlined above.

### CCPA/CPRA (California Users)

We aim to comply with California privacy requirements, including:

- Right to know what data we collect.
- Right to delete personal information.
- Right to opt out of sale or sharing where applicable.
- Non-discrimination for exercising privacy rights.
